Wie funktioniert der Crawler?

  • Nach Eingabe der URL wird die ganze Seite gecrawled
  • Collecten alle Links auf der Seite und packen alle Links in eine Liste
  • Wir bleiben in der selben Domain
  • Suchen die ganzen Seiten auf Basis des regulären Ausdrucks nach Email-Adressen ab
  • Passt eine Email-Adresse zu unserer selbst erstellten Prioritäten Liste ("Impressum", "Info" etc.), wird diese Adresse in einer Liste gespeichert

Anker entfernen

 

    private String removeAnchorFromLink(String foundUrl) {
        if (foundUrl.contains("#")) {
            return foundUrl.split("#")[0];
        } else {
            return foundUrl;
        }
    }

Workflow

  • Docker
  • Nach Push auf Github wird über Dockerhub ein neues Image erzeugt
  • Bereits laufende Container werden alle 5 Minuten nach Updates überprüft
  • Sollte ein Update verfügbar sein, wird der Container über Dockerhub geupdated
  • Somit dauer ein Update vom Push bis zum Release ca. 10 Minuten
